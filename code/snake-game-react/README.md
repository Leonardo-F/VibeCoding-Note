# 掌机贪吃蛇 🐍

一个基于 React 开发的复古掌机风格贪吃蛇游戏，具有多种特殊食物效果和精美的视觉设计。

## 游戏特色

### 🎮 复古掌机风格
- 精心设计的掌机外观界面
- 经典的 D-Pad 方向键布局
- 霓虹灯效的视觉体验
- 响应式设计，支持桌面和移动设备

### 🍎 多种食物类型

| 食物 | 形状 | 颜色 | 分数 | 效果 | 出现概率 |
|------|------|------|------|------|----------|
| 普通食物 | 圆形 | 绿色 | 10 | 无 | 50% |
| 加速食物 | 三角形 | 红色 | 15 | 蛇速加快 | 15% |
| 减速食物 | 方形 | 青色 | 5 | 蛇速减慢 | 15% |
| 双倍积分 | 星形 | 金色 | 20 | 10秒内双倍积分 | 10% |
| 缩短蛇身 | 菱形 | 紫色 | 25 | 缩短3节蛇身 | 10% |

### 🎯 游戏机制
- **速度系统**: 吃到加速或减速食物会改变蛇的移动速度
- **双倍积分**: 吃到星形食物后10秒内所有食物积分翻倍
- **蛇身管理**: 吃到菱形食物可以缩短蛇身，帮助在危险情况下生存
- **最高分记录**: 自动保存游戏最高分
- **实时反馈**: 显示当前速度、效果状态和最近获得的效果

## 游戏玩法

### 控制方式

#### 键盘控制
- **方向键 (↑↓←→)** 或 **WASD**: 控制蛇的移动方向
- **空格键**: 开始游戏 / 重新开始

#### 触屏/鼠标控制
- 点击屏幕下方的 **D-Pad** 按钮控制方向
- 点击红色的 **开始** 按钮开始游戏

### 游戏规则
1. 按空格键或点击开始按钮启动游戏
2. 控制蛇移动，吃掉食物获得分数
3. 不同颜色的食物有不同的效果和分数
4. 碰到墙壁或自己的身体游戏结束
5. 尽可能获得更高分数！

### 策略提示
- **速度管理**: 合理利用加速和减速食物，在需要快速移动时吃红色三角形，需要精确控制时吃青色方形
- **双倍积分**: 吃到金色星星后，尽快多吃食物获得双倍分数
- **蛇身控制**: 当蛇身过长难以控制时，寻找紫色菱形食物缩短蛇身
- **优先级**: 根据当前情况选择合适的食物，不要盲目追求高分

## 技术实现

### 技术栈
- **React 18**: 使用 Hooks 进行状态管理
- **Vite**: 快速的开发构建工具
- **CSS3**: 实现动画和视觉效果

### 核心功能
- **游戏循环**: 使用 `setInterval` 实现蛇的移动
- **碰撞检测**: 检测蛇与墙壁、自身和食物的碰撞
- **食物生成**: 基于概率权重的随机食物生成算法
- **状态管理**: 使用 React Hooks 管理游戏状态
- **键盘事件**: 监听键盘输入并转换为方向控制

## 安装和运行

### 环境要求
- Node.js 16.0 或更高版本
- npm 或 yarn 包管理器

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```
